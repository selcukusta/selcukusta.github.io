Hakkımda

Herkese selamlar!

1987 İstanbul doğumluyum. Okan Üniversitesi Bilgisayar Mühendisliği Yüksek Lisans programından mezun oldum. Aktif olarak yazılım ağırlıklı teknik yazılar yazıyor ve teknoloji topluluklarında gönüllü konuşmacı olarak görev alıyorum.

9 yıldır profesyonel olarak yazılım geliştirici olarak teknoloji dünyasının içerisindeyim. Şu anda da Rasyotek İnsan Kaynakları Bilişim A.Ş.‘de Yazılım Geliştirme Müdürü olarak görev yapıyorum.

İş hayatıma Bilge Adam Bilişim Grubu’nda Yazılım Geliştirme Uzmanı / Danışman pozisyonu ile başladım. Yaklaşık 2.5 sene kadar .NET programlama ve SQL Server programlama konularında proje geliştirdim, danışmanlık hizmetinde bulundum.

2012 - 2014 yılları arasında Lynx S.p.A’de Yazılım Uzmanı olarak mesleğimi icra etmeye devam ettim. Finans ve sigorta uygulamaları üzerinde yazılım geliştirdim. Vakıfbank, ING Bank, BNP Paribas Cardif gibi müşterilerle çalıştım.

Kasım 2014 – Nisan 2015 arasında askerlik görevimi yerine getirdikten sonra Hürriyet Gazetecilik ve Matbaacılık A.Ş.’de Kıdemli Yazılım Uzmanı, sonrasında ise Yazılım Teknik Takım Lideri olarak görev aldım. hurriyet.com.tr ve çevresinde gelişen dikey portallerin kesintisiz bir şekilde yayın hayatına devam etmesi, yeni uygulamaların mimarilerinin kurgulanması ve SDLC süreçleri görev ve sorumluluk alanlarım içerisinde yer almaktaydı.

Mart 2018 - Ağustos 2018 tarihleri arasında KoçSistem‘de, Ağustos 2018 - Nisan 2019 tarihleri arasında ise Demirören Teknoloji‘de Yazılım Mimarı olarak görev aldıktan sonra; son olarak mevcut pozisyonum ile Rasyotek İnsan Kaynakları Bilişim A.Ş.‘de görevime devam etmekteyim.

2010 yılından bu yana ise MCP, MCPD, MCTS sertifikasyonlarına sahibim.

Temel olarak ilgi alanımdaki teknolojiler ise aşağıda sıraladığım gibi;

.Net, .Net Core, C#, Python

Platforms:

Microsoft Azure, Amazon Web Services, Pivotal Cloud Foundry

Database&In-memory Data Systems:

Redis, Memcache, SQL Server, MongoDB, Elasticsearch, RabbitMQ

Technologies:

Linux, ASP.NET Core, RESTful Services, Python Tornado Web Framework, Varnish, Nginx

Tools:

Teamcity, Octopus, Travis CI, Jira, Crucible, Sonarqube, Selenium, Git, TFS, Docker, Docker Swarm, Kubernetes, Concourse, Vagrant, Packer, Ansible, Vault, Terraform
